Typography

Heading Font:[Cinzel]

Heading Example

I chose this font for the headings because I want my site to have a clean, crisp, look and feel to it. I felt like this font is bold enough for the headings, but also blends well with the colorpallette of this site.

Subheading Font:[Pathway Gothic one]

Sub heading Example

I chose this font for the subheadings because I wanted them to contrast well enough with the headings so that the reader could tell them apart.
